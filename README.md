# Inflation_Stock_Returns_Analysis
This is my year 2 Data Science project for the module BEE2041_A_2_202526. I explore the link between inflation metrics and stock returns in the USA.


##Library and Version requirements:

Python: 3.12.0
Pandas: 3.0.1
Matplotlib: 3.10.8
Seaborn: 0.13.2
Scipy: 1.17.1
Statsmodels: 0.14.6
Yfinance: 1.2.0
Fredapi: N/A
